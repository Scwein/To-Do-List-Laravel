# To-Do List API & Web App

Простое веб-приложение для управления списком задач (To-Do List) с чистым интерфейсом и полноценным **RESTful API**. Проект создан для демонстрации навыков разработки на PHP/Laravel.

## О проекте

Этот проект представляет собой классический менеджер задач, который позволяет пользователям управлять своими делами. Ключевая особенность — наличие как удобного веб-интерфейса, так и готового API, что позволяет интегрировать управление задачами в другие приложения.

## Технологии

- PHP 8.0+
- Laravel 12
- MySQL / SQLite
- Bootstrap 5

### Основные возможности

*   **Создание, чтение, обновление и удаление (CRUD)** задач.
*   Чистый и интуитивно понятный веб-интерфейс.
*   **Полноценный RESTful API** для программного управления задачами.
*   API возвращает ответы в формате JSON.

## API Документация

API позволяет выполнять все основные операции со списком задач. Базовый URL для всех запросов: `/api/tasks`

### Эндпоинты

| Метод   | Эндпоинт           | Описание                            |
|---------|--------------------|-------------------------------------|
| GET     | /api/tasks         | Получить список всех задач          |
| GET     | /api/tasks/{id}    | Получить данные конкретной задачи   |
| POST    | /api/tasks         | Создать новую задачу                |
| PUT     | /api/tasks/{id}    | Обновить существующую задачу        |
| DELETE  | /api/tasks/{id}    | Удалить задачу                      |
